# ClipMore

ClipMore is an innovative Discord bot designed to streamline social media campaign management and creator compensation. The bot allows users to submit social media posts, automatically tracks engagement metrics across platforms, and provides campaign managers with tools to efficiently manage payouts based on view counts.

## Development Setup

### Prerequisites
- Node.js
- PostgreSQL
- Discord Bot Token
- Social Media API Keys

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/clipmore.git
cd clipmore

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Start development servers
npm run dev
```
